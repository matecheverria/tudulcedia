# Operacion diaria Supabase

## Accesos

- Catalogo publico: `/`
- Panel privado: `/panel-td-privado-2026.html`
- Dashboard principal: `/admin-dashboard-supabase-v3.html`
- Respaldo antiguo: `/admin-dashboard.html`

## Rutina

1. Abrir el panel privado.
2. Entrar al dashboard principal.
3. Revisar pendientes.
4. Confirmar disponibilidad.
5. Actualizar estado.
6. Actualizar pago.
7. Contactar al cliente.
8. Revisar ingresos activos.

## Estados

Pedido:

- Pendiente
- Confirmado
- En preparacion
- Listo para retiro/entrega
- Entregado
- Cancelado

Pago:

- Pendiente de comprobante
- Comprobante recibido
- Pagado
- Pago al retirar
